# Rang Salaka (React)

A responsive React implementation of the Rang Salaka grid with interactive selection, 18-step traversal, and fuzzy matching against `ans_compare`. Shows the corresponding answer and description in a modal.

## Run

```bash
npm install
npm run dev
```

Open the printed local URL (default: http://localhost:5173). Click any cell to generate a sequence and see results. Links at the bottom open `../rang_salaka.html` and `../rangsalaka-final.html`.

## Build

```bash
npm run build
npm run preview
```
